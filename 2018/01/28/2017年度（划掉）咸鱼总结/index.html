2017年度（划掉）咸鱼总结

发表于 2018-01-28 | 分类于 life | 本文阅读量次
字数统计: 539字 | 阅读时长 ≈ 2分钟

2016年末的时候好像也是写了一个这样的总结，不过好像没有放在博客里，2016年末写的年度期望可以说是2017年里不会完成的事儿大集合了呢。

前阵子一直咸鱼，加上期末复习季，很多学习进度都搁置了，内心无比愧疚，接下来的寒假希望能好好地学习，好好睡觉！！

简单地对2017做一个小小总结吧，毕竟这一年也是算是有很大改变的一年（至少是在人生轨迹上）。

三月初的春招通通gg，人生轨迹算是也或多或少因此受到了很多的影响，而且同时改变了我大一上对于两大团队的看法，夏令营在贵创遇到了可爱的mentor（露露好强），写了很多ddd型代码（笑），最后也算是终于加入贵创啦w。

咸鱼的时间大概占了全年的3/4吧，个人感觉，大概总结也就这样了吧（好像又不想写了。。。。。。有空再补一下吧

arch Linux用户 => mac用户，ducky => poker ，sony 1ADAC => 用来屏蔽周边废话的1000x，6s => 8p （啊啊啊啊今年要好好赚钱！）

我好像不记得自己大一下在做什么了，姑且将其算做咸鱼吧，从暑假开始写了一个多月的代码，感觉很有意思，

在2018年初列一下flag吧：（啊啊希望今年不要鸽自己了）

  1. 在暑假的时候希望能找到实习吧（这几天微校园一直提醒消息，加权凉凉，终于不再想着要拯救加权什么的了）
  2. 寒假的时候写课设，读读书，把之前欠着的书读一下啦w
  3. 开春的时候还是继续写web，组件库什么的应该等露露回来就鸽不了啦，写个博客主题，做个像mikuTap一样有趣的东西。
  4. 学会玩四阶五阶魔方！（这原本是2017年度计划的orz）
  5. 努力赚钱，今年期待可以把单反升级一下下！！（这大概可能难难哦！）
