FCC-Responsive Design with Bootstrap

发表于 2017-03-16 | 分类于 FCC-learning

最近又重新开始在FCC上做题目，上次把HTML5 and CSS完成了，这次继续往下做一章，就是用来记录一下记不太清的东西还有不太懂的知识点．

Use Responsive Design with Bootstrap Fluid Containers

You can add Bootstrap to any app by adding the following code to the top of your HTML:

Make Images Mobile Responsive

add the class Make Images Mobile Responsive to img element

Center Text with Bootstrap

add the class text-center to h2 element

Create a Bootstrap Button

Bootstrap has its own styles for button elements.

add the class btn to button element

Create a Block Element Bootstrap Button

to make your buttons stretch to fill your page’s entire horizontal space

add the class btn-block to button element

Taste the Bootstrap Button Color Rainbow

add the class btn-primary to button element

Note that these buttons still need the btn and btn-block classes.

Call out Optional Actions with Button Info

The btn-info class is used to call attention to optional actions that the user can take.

add the class btn-info to button element

Note that these buttons still need the btn and btn-block classes.

Warn your Users of a Dangerous Action

to notify users that the button performs a destructive action, such as deleting a photo

add the class btn-danger to button element

Use the Bootstrap Grid to Put Elements Side By Side

add col-md-4 row classes

Use Spans for Inline Elements

Create a Custom Heading

Bootstrap uses a responsive grid system, whick makes it easy to put elements into rows and specify each element’s relative width. Most of Bootstrap’s classes can be applied to a div element.

Add Font Awesome Icons to our Buttons

You can add Font Awesome to any app just by including it by adding the following code to the top of your HTML:

Add Font Awesome Icons to all of our Buttons

FontAwesome is a convenient library of icons. These icons are vector graphics, stored in the .svg file format.

And these icons are treated just like fonts.

Style Text Inputs as Form Controls

Give your form’s text input field a class of form-control.

Give your form’s submit button the classes btn btn-primary.

Also give this button the Font Awesome icon of fa fa-paper-plane.

Line up Form Elements Responsively with Bootstrap

Nest both your form’s text input and submit button within a div with the class row. Nest your form’s text input within a div with the class of col-xs-7. Nest your form’s submit button in a div with the class col-xs-5.
