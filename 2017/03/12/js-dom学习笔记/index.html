什么是dom

说白了DOM就是浏览器为js提供的一系列接口，通过window.docement提供，通过这些接口我们可以操作web页面．

推荐书籍：＜js dom编程艺术>　很好看，谁看谁知道

但是注意dom不是编程语言，只是文档对象的模型，该模型是独立于编程语言的，比如我们在py中也可以操作dom：

所以Web前端常讲的DOM API (web 或 XML 页面) = DOM + JS (脚本语言)

dom创建

节点（node）通常对应一个标签一个文本，或者一个html属性，DOM节点有一个nodeType来表示当前元素的类型，是一个整数：

  2. Attribute，属性
  3. Text，文本

dom更改

js dom中是没有insertAfter()这个操作方式的，所以可以选择自己写一个insertAfter()函数，会比较方便．

实现思路：先判断目前的元素节点是否是该父节点的最后一个子节点，如果是的话，直接appendChild即可；如果不是最后一个字节点．则获取想插入的元素节点的后一个元素节点的位置，此处使用的是nextSibling，然后使用insertBefore
即可．
