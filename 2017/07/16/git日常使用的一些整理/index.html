git日常使用的一些整理

发表于 2017-07-16 | 分类于 git

虽然日常使用git，但是对它很多时候还是常常无从下手甚至非常担心一个误操作就导致之前的代码都前功尽弃。以前碰到过也解决了很多git的问题，但是经常再碰到的时候就又忘记解决方法，然后又要去查，非常浪费时间，所以就以后都记录一下好啦。（顺便就可以经常更新博客了嘻嘻

git比较本地仓库和远程仓库的差异

原文链接：https://github.com/KylinGu/Test/blob/master/GitLearningExp.md

在整理git仓库的时候，发现了自己写c语言实验时候的代码，但是不知道为什么本地的代码库是更新到最新的cpractice8，但是远端只有cpractice6，进行了git pull和git push似乎都没有任何的效果，而且本地也无法clone远程仓库，比较本地仓库和远程仓库的差异的时候也是没有任何发现，这个问题目前还没有解决。

在pull之前，可以先比较本地仓库和远程仓库之间的差异

  1. 添加需要比较的远程仓库：git remote add foobar git@github.com:${username}/${reponame}.git

  2. 取回foobar的内容，fetch不会修改本地的内容：git fetch foobar

  3. 比较本地分支和远程分支之间的差异：git diff master foobar/master

  4. 远程分支已经修改，本地未同步的变更：git diff HEAD...origin/master

  5. 本地分支已经修改，远程未同步的变更：git diff origin/master...HEAD
