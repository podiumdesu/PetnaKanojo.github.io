不让组件渲染

当你在某些情况下不想让组件渲染的时候，请使用return null这样就可以隐藏它啦

当从组件的render()中返回null的时候，是不会影响组件的生命循环方法的。这就意味着componentDidMount和componentWillUnmount方法仍然是会被调用的哦。
