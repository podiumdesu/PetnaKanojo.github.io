虽然在React元素中处理事件和在DOM元素中处理事件是相同的，但是还是有一些语法上的问题需要注意：

  1. React事件使用驼峰命名(camelCase)，而不是小写
  2. 使用函数作为事件处理器，而不是一个字符串
