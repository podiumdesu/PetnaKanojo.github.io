使用form获取让用户提供数据

  • action（一个url）:想让用户提交的数据提交到什么地方
  • method（是用get方式提交还是post方式提交）

GET vs. POST

  • get更安全，不对服务器上的数据进行改动，每一次操作的结果是相同的
  • post每次对服务器上的内容有修改
  • get的数据会缓存，post数据不缓存
  • 数据传输：post不会把数据放在路径里，而是进行编码，放在http的body里提交上去。http传输是分两拨的。
  • 编码：get放在url里，post放在body里。

url编码

  => %20 (空格)
! => %21
" => %22
# => %23
$ => %24
% => %25
& => %26
' => %27
...

输入验证

  • 安全性、使用上要提示用户输入正确的信息

  • required 必填项，不需要任何值

  • minlength、maxlength
  • 正则表达式pattern，提示输入手机号。1\d{10} 首位是1，剩下十位

type：用于验证（内置）

novalidate：不要使用原生的验证，使用js自己写好的验证

回车提交
