行级格式化上下文

  • 主要记住base line

  • linebox内的盒子摆放

vertical-align

  • 定义盒子所处的行盒(line box)的垂直对齐关系
  • 取值：baseline | sub | super | top | text-top | middle | bottom | text-bottom | |
  • 百分比相对于元素自身的行高
  • 初始值 baseline

lists style

  • display: list-item 会生成 Principle Block Box 和 Marker Box
  • Marker Box 的内容和位置可以通过 list-style 系列属性指定

list-style-type

  • none
  • disc
  • circle
  • square
  • decimal
  • lower-roman
  • upper-roman
  • lower-greek
  • lower-latin
  • upper-latin
  • armenian
  • georgian
  • lower-alpha
  • upper-alpha

list-style-image

background

  • 每个盒子都可以设置背景。

  • background-color

  • background-image
  • background-repeat 一个容器里面设置图片的平铺与否，默认是repeat,还有round和space
  • background-position　　设置从哪里开始显示，默认是０
  • background-attachment　　

background-size

border-radius

（圆角）

  • border-radius:5px
  • 可以指定四个方向
  • 可以使用百分数
