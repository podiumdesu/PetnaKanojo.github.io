全局属性

  • 几乎所有的元素都适合的全局属性

accesskey&tabindex

  • 为了可操作性

id,class&style

  • id 保证唯一性
  • class 多用在 CSS
  • style 指定内联样式

contenteditable & spellcheck

  • contenteditable使页面上的内容可以被修改
  • spellcheck是拼写检查

语言lang & dir

  • lang是指定语言
  • dir是指定书写顺序

title

  • 鼠标移到缩写上时会把更多内容放在title属性里
  • 但是有一点延时，会用tips来代替title

hidden

  • 把元素隐藏起来，可能是这个元素在某个点击后会再展示出来
  • 以往会用display:none，要想再显示出来，可以用block或inline等，但是要做判断。
    用hidden不需要考虑这些

ARIA

  • role表示充当的是什么角色

无障碍性

  • 或可访问性，Accessibility
  • 确保任何人都有办法获取放在网页上的媒体内容
  • 不让身体、心理或技术上的问题成为获取信息的障碍

提升无障碍性

  • 为 img 提供 alt 属性
  • noscript:有些情况下不支持js，一旦脚本被禁用了，可以用noscript来告诉用户怎么启用，或者怎么找到替代版本。
  • input 和 label 对应
  • 图形验证码与语音验证码
  • 文字和背景有足够对比度
  • 键盘可操作

web开发者应该做的事情

  • https://www.w3.org/TR/WCAG20/
  • https://www.w3.org/TR/wai-aria/

语义化

  • HTML中的元素、属性及属性值都拥有某些含义
  • 开发者应该遵循语义来编写HTML
语义化的重要性
  • 提升代码可读性、可维护性
  • 搜索引擎优化
  • 提升无障碍性

meta标签

data-* （没懂）

  • html里标签

microdata（搜索引擎优化）

  • HTML 5 中的一个规范
  • 在 HTML 中通过属性嵌入格式化数据
  • 提供给搜索引擎、浏览器(插件)使用
