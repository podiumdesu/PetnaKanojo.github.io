MacTex配置fontawesome

2017-05-23

最近报名了一个hackinit()的活动，然后需要写一份简历，去github上捞了一波模版，然而对于一个从来没用过latex的小白来说，最麻烦的怕不就是本地环境的搭建了w

捞的resume模版在这里！，是联创前队长写的一份uniquecv。

下面开始配置：

安装MacTex

听很多人说这个软件是mac下比较好用的latex工具，所以就下载了，对于没接触过latex的小白来说，直接点击下载那个完整版，就是2.7G的那个文件

这里是下载链接

打开以后会得到一大堆的东西

打开taxshop

之后的编译等等都是在这个软件下进行的。具体的latex语法什么的就不细说了。

配置中文

  1. 左上角将latex选为xelatex

  2. 打开texshop的「偏好设置」将「字符编码」改为utf-8，记得保存文件的时候也看一下是否是以utf-8编码的哦w

  3. 前往系统的「应用软件」->「字体集」，查看你需要的是什么字体。
    如图上所示：Songti SC

  4. 测试一下w：

此时应该就可以显示出中文啦
