一点想说的

发表于 2017-02-04 | 分类于 reading | 本文阅读量次
字数统计: 189字 | 阅读时长 ≈ 1分钟

最近在知乎上看到了一篇文章，里面的很多内容我都不能更赞成了。摘录下来啦

原文链接：爱情到底要不要结果

我喜欢你是因为喜欢你，更是因为我喜欢你有梦想的样子。爱情的最高级形式，其实是自由，你去追你心中的那团火，我也去。说爱情永恒，其实是说，我在精神上永远同你缔结，我是你始终如一的支持者，因为我们认领了自己人生的目标，并各自为之努力。

剩下的想说的实在不知道怎么措辞，以后再补吧，总之我真的很赞同这篇文章的观点，真的太相符了。
