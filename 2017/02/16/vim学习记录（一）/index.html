第一级–存活

啊哦，下面是一些命令，可以在Normal模式下幸存下来。

  • x -> 删当前光标所在的一个字符（应该是和del效果是一样的）
  • dd -> 删除当前行，并将删除的行保存在剪贴板里
  • hjkl -> 使用其来移动光标
  • :help<command> -> 显示相关命令的帮助，使用:q退出

注意: 在vim的normal模式下，所有的键就是功能键了！！！

第二级–感觉良好

注意: 所有的命令都需要在normal模式下使用，如果不知道现在在什么模式，就多使用几次esc

  1. 各种插入模式

    • a -> 在光标后插入
    • o -> 在当前行后插入一个新行
    • O -> 在当前行前插入一个新行
    • cw -> 替换从光标所在位置后到一个单词结尾的字符

  2. 简单的移动光标

    • 0 -> 数字零，到行头
    • ^ -> 到本行地一个不是blank字符的位置（blank即空格，tab，换行，回车等）
    • $ -> 到本行行尾
      g_ -> 到本行最后一个不是blank字符的位置
      /petnakanojo -> 搜索petnakanojo的字符串（使用n键到下一个）
